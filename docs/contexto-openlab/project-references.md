## Cómo Usar Este Archivo

Busca proyectos relevantes para el cliente actual por:
- **Sector:** ¿Hay algún proyecto en el mismo sector o uno similar?
- **Tipo de proceso:** ¿El proceso que quiere automatizar el cliente es parecido a algo que ya hicimos?
- **Tamaño de empresa:** ¿Hay un cliente de tamaño comparable?

Si no hay ninguna referencia relevante, indica "Sin referencias directas en este sector" en el assessment.

## Proyectos Referenciables

### Iberostar — Observatorio de Tendencias IA
| Aspecto | Detalle |
|---------|---------|
| **Sector** | Turismo / Hotelería |
| **Tipo de proceso** | Scouting de tendencias, inteligencia de mercado |
| **Descripción** | Observatorio automático de tendencias IA para innovación hotelera. Corre semanalmente en Azure, investiga nuevas tecnologías, valora relevancia para hoteles, entrega informe a Google Drive + notificación por email. |
| **Modo de ejecución** | Headless/Scheduled (cron semanal) |
| **Resultado** | De 0 informes proactivos a 1 informe automatizado por semana |
| **Referenciable públicamente** | Consultar antes de nombrar al cliente |

### Market Due Diligence — VCs
| Aspecto | Detalle |
|---------|---------|
| **Sector** | Venture Capital / Inversión |
| **Tipo de proceso** | Deal screening, due diligence de mercado |
| **Descripción** | Agente de 7 fases autónomas: deal screening contra criterios del fondo, validación TAM/SAM/SOM, descubrimiento de competidores, análisis de tendencias, red flags, recomendación de inversión. |
| **Modo de ejecución** | Interactivo (bajo demanda) |
